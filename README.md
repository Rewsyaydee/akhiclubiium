The website is designed with a modern, "blueprint" aesthetic (Cyanotype theme) to reflect our mission of architectural character building.

## ğŸš€ Website Features

This single-page application (SPA) is built to be lightweight, responsive, and visually engaging.

### 1. **Cinematic Hero Section**
- Immersive entry with a noise/grain overlay for a textured feel.
- Custom cursor implementation for a unique browsing experience.
- Responsive typography that scales from mobile to desktop.

### 2. **Interactive Archive Gallery**
- Horizontal scrolling gallery showcasing past expeditions (Bright Future Project, Fateh, Aulad).
- **Hover Effects**: Thumbnails rotate through album precursors on hover.
- **Lightbox Integration**: A custom JavaScript lightbox allows users to view full albums of photos and videos without leaving the page.

### 3. **Organizational Structure (The Lineup)**
- A 3D flipping card interface for the organizational chart.
- Displays High Committee and various bureaus (Human Resources, Entrepreneurship, Multimedia, Aulad, Training & Development, Fateh).
- Cards flip to reveal member photos and additional details.

### 4. **Partnership & Outreach Form**
- Integrated contact form for schools to request collaborations.
- Powered by `Formsubmit.co` for backend-less email handling.
- Captures institution details and program objectives directly.

### 5. **Design System**
- **Cyanotype Theme**: Utilizing deep blues (`#001a2c`, `#004a7c`) and paper whites (`#f4f9fc`) to mimic architectural blueprints.
- **Animations**: Scroll-triggered reveal animations using `IntersectionObserver`.
- **Typography**: Uses 'Inter' for body text and 'JetBrains Mono' for technical aesthetic accents.
